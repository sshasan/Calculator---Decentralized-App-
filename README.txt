
############################## @@@@@ Distributed Calculator ###########################################
* This is a simple Distributed Application(Dapp), running on ethereum Blockchain.
* To use this dapp on local client-
            -> npm install -g ethereumjs-testrpc
            -> testrpc 
            -> mkdir Calculator 
            -> cd Calculator
            -> npm init 
            -> npm install ethereum/web3.js --save 
* Copy the .sol file into the remix browser and copy ABI and address and paste it into the index.html in designated place.
* Use the Application.
